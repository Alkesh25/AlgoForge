# 🚀 AlgoForge


AlgoForge is a full-stack DSA practice platform with Algorithm Visualizer where users can track progress, attempt quizzes, and follow a structured roadmap.


---


## 🌐 Live Demo

👉 https://whimsical-gumdrop-ba0b70.netlify.app


---


## ⚙️ Tech Stack


- Frontend: React, Tailwind CSS
- Backend: Node.js, Express.js
- Database: MongoDB
- Authentication: JWT
- Deployment:
  - Frontend → Netlify
    
  - Backend → Render

---


## ✨ Features


- 🔐 User Authentication (Login / Signup)
- 📊 Progress Tracking System
- 🧠 Quiz System (Easy / Medium / Hard)
- 🛣️ DSA Roadmap Navigation
- 📈 Dashboard with analytics
- 🎨 Clean and responsive UI
